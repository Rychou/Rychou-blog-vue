文章的密码验证

今天写博客的时候想着要些一些日记之类的博客的话，如果没有密码验证那就不叫日记了，所以在网上找了一下方法，下面是我根据别人的改进的js代码,加在文章中可以实现密码验证
直接附上源码

<script>

while(1){
if("160423"==prompt("Please input password"))
{
    break;
}
else
{
    alert("输入错误");
}
}

</script>

不过这个代码还是不完善的，比如说如果你的主题主页不是单纯的列表页的话，那么你在主页就会出现这个密码验证。所以说不完善，博主还在想办法完善中。
